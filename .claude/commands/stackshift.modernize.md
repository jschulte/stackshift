Invoke the modernize skill for Brownfield Facelift mode.

This is the **MODERNIZATION STEP** for Brownfield projects that selected the Facelift option.

Runs after Gear 6 to:
- Upgrade all dependencies to latest versions
- Fix breaking changes with spec guidance
- Improve test coverage to 85%+
- Update specs to reflect changes
- Generate upgrade report

Perfect for modernizing legacy apps without full rewrites.

Use the Skill tool with skill="modernize".
